# Implementación del Algoritmo DFS
Este directorio contiene una implementación en C++ del algoritmo DFS (Búsqueda en Profundidad) para un grafo.

## Funcionamiento del Algoritmo
DFS explora un grafo en profundidad antes de retroceder. Es útil para recorrer grafos, detectar ciclos y contar componentes conectados.

## Compilación y Ejecución
Para compilar el archivo `dfs.cpp`, puedes usar el siguiente comando:

```bash
g++ -o dfs dfs.cpp

./dfs
