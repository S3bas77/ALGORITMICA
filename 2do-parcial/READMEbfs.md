# Implementación del Algoritmo BFS
Este directorio contiene una implementación en C++ del algoritmo BFS (Búsqueda en Anchura) para un grafo.

## Funcionamiento del Algoritmo
BFS explora un grafo nivel por nivel. Es útil para encontrar la distancia mínima entre dos nodos, verificar conectividad y generar árboles de cobertura mínima.

## Compilación y Ejecución
Para compilar el archivo `bfs.cpp`, puedes usar el siguiente comando:

```bash
g++ -o bfs bfs.cpp

./bfs
